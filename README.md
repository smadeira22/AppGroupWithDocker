# Geography Game Appp

## Clone the repository

- git clone 

## Install dependencies

- For back end:
  - cd into server 
  - npm install

- For front end:
  - cd ../client 
  -npm install

## Open Docker Application

## Start Docker containers

- cd .. 
- docker compose up

## Website urls

- The API service runs on http://localhost:3000/ 
- The client can be accessed at http://localhost:8080/

## Stop application

- docker compose down
